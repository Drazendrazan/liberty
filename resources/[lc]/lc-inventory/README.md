## Features
- Item crafting
- Weapon attachment crafting
- Stashes (Personal and/or Shared)
- Vehicle Trunk & Glovebox
- Weapon serial number
- Shops
- Item Drops

## Installation

### Manual
- Download the script, rename it to `aj-inventory` and put it in the `[qb]` directory.
- Start the server
